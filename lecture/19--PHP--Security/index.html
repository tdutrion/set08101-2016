PHP supported versions

Passwords

Do not use md5, sha1 or similar for passwords.

In PHP 5.5 +

  • password_hash
  • password_verify

password_compat from Anthony Ferrara can be used as a polyfill for PHP 5.3.7+

Database access

  • Use PDO or mysqli, no mysql_* functions (deprecated, no database resource required explicitely)
  • Use prepared statements

In a nutshell

  • Never trust [user] input
  • Always filter output
  • Keep it simple - complexity increases the likelihood of security holes
  • Principle of least privilege
  • Don't write your own crypto

Random values

PHP rand function does not provide completely random values.

In PHP 7+, you can use random_int and random_bytes

In PHP 5.2+, you can use the polyfill random_compat from ParagonIE.
