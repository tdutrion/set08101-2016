CSS media queries

<head>
    <style>
        .wrapper {
            border: solid 1px #666;
            padding: 5px 10px;
            margin: 40px;
        }
        @media screen and (max-width: 600px) {
            .wrapper {
                background: #F9C;
            }
        }
    </style>
</head>
<body>
    <div class="wrapper">pink when viewing area < 600px</div>
</body>

Using bootstrap

Design for developers

What is it?

  • A set of css classes for positions and styles
  • Optional javascript libraries (jQuery plugins)
  • Icons

Philosophy

"Easy to use"

Just use classes to enable styles to your elements:

<button class="btn btn-default">Default</button>

Positionning

Bootstrap is providing a 12 columns grid system.

It also supports nesting rows.

Layout

A page contains a container, responsible for the overall width of the block.

containers contain rows, and rows contain columns.

<div class="container">
    <div class="row">
        <div class="col-lg-3">Menu</div>
        <div class="col-lg-9">Content</div>
    </div>
</div>

See the documentation for more features.

My favourites are tables and buttons.
