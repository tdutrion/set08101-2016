Create "dynamic" lists

Programmes page

<?php
$programmes = [
  'Program 1',
  'Program 2',
  'Program 3',
];
?>
<body>
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="staff.html">Staff list</a></li>
    <li><a href="programmes.html">Programmes</a></li>
  </ul>
  <h1>Edinburgh Napier University</h1>
  <p>This contains a list of programmes</p>
  <ul>
    <?php foreach ($programmes as $program): ?>
      <li><?php echo $program; ?></li>
    <?php endforeach; ?>
  </ul>
</body>

Introduce objects and classes

Classes

A class is a type of object, a data structure.

Objects

An object is a variable, instance of a specified class.

Example: Andrew Cumming is an instance of Lecturer, Andrew being the object and lecturer the class.

Application: staff page (1/5)

<table>
  <tr>

$lecturers = [

Application: staff page (5/5)

Add a lecturer with no office nor phone (3/3)

<?php
class Lecturer
{
  public $surname;
  public $firstname;
  public $office;
  public $phone;

  public function __construct($surname, $firstname, $office = null, $phone = null)
  {
    $this->surname = $surname;
    $this->firstname = $firstname;
    $this->office = $office;
    $this->phone = $phone;
  }
}
?>
