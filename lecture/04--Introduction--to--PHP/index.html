Static versus dynamic

Static

  • Slow to update
  • Not personalised to for the specific visitor
  • Good for showcasing websites that do not change often

Dynamic

  • Requires to know how to develop static websites
  • Allows to retrieve data from other sources (databases, webservices)
  • Allows contributions from visitor/user

PHP: basics (syntax and structures)

Syntax: PHP tags

PHP code is written in text files. To enable the PHP interpretation, one use the PHP tags. 

<?php
?>

NB: PHP use to have other opening tags (short open tags, asp tags), please do not use them anymore (except for the echo shortcut ).

NB: the closing tag is not required if there is nothing else than PHP at the end of the file. It is consider as a good practice not use them when they are not required.

Instructions

An instruction in a programming language is a set of characters that correspond to the machine commands.

In PHP, instructions are separated with a semi-column. That said, closing tags and end of files automatically imply the end of the instruction, therefore the semi-column is not required in these cases.

<?php echo date('d/m/Y'); ?>
<?php echo date('d/m/Y') ?>
<?php echo date('d/m/Y')

Comments

PHP supports 'C', 'C++' and Unix shell-style (Perl style) comments.


<?php
  echo 'This is a test'; // This is a one-line c++ style comment
  /* This is a multi line comment
    yet another line of comment */
  echo 'This is yet another test';
  echo 'One Final Test'; # This is a one-line shell-style comment

  //======================================================================
  // CATEGORY LARGE FONT
  //======================================================================

  /* Title Here Notice the First Letters are Capitalized */

  /*
   * This is a detailed explanation
   * of something that should require
   * several paragraphs of information.
   */
https://secure.php.net/manual/en/language.basic-syntax.comments.php

Types

PHP supports 8 primitive types:

  • scalar types
    • boolean
    • integer
    • float (historically named double)
    • string
  • compound types
    • array
    • object
  • special types
    • resource
    • NULL

Everything is in the documentation if you want to learn more.

Variables


<?php
  $var = 'Bob';
  $Var = 'Joe';
  echo "$var, $Var";      // outputs "Bob, Joe"

  $4site = 'not yet';     // invalid; starts with a number
  $_4site = 'not yet';    // valid; starts with an underscore
  $täyte = 'mansikka';    // valid; 'ä' is (Extended) ASCII 228.

Constants
